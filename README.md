###### biqugeCrawler

这是一个爬取笔趣阁小说(biquge.tw)的程序。只需要提供小说的URL，就可以按照设置的参数把每个章节下载并保存为单独的HTML格式。配合Sigil程序，可将下载的HTML文件生成为EPUB电子书格式。



###### 参数的获取方法

先搜索到想要的书，打开阅读第一章。这时地址栏中的https://biquge.tw/book/9002/286409.html 两个数字即为我们需要的参数，分别把它们按顺序填入到程序中。

###### 目前的问题以及未来可能发生的改变

| 问题                                                         | 解决                                                         |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| ~~需要手动输入bookid与index~~                                | ~~只需要提供网络地址，无需手动输入参数~~                     |
| ~~网址不连贯可能导致只能下载前部分章节~~                     | ~~将配合网址递增以及解析界面超链接两种方式适时跳过缺失的章节~~ |
| 输入参数时需要改动源码                                       | 引入配置文件，无需改动源码                                   |
| 由于作者书写习惯导致目录中无法显示章节标题，部分作者习惯只在标题处写第XXX章，而将标题放在正文第一行，导致使用Sigil生成目录时无法显示真正的章节标题 | 采用多种内容解析方式，由用户自主选择标题样式                 |
| 读书软件目录功能存在显示/不显示章节数两种风格，可能会导致章节数缺失或者重复显示 | 同上                                                         |
| ~~需要人工确定章节数量~~                                     | ~~由程序自行获取~~                                           |
| 单线程网络请求导致浪费大量时间                               | 引入多线程                                                   |
